(문제12)
자연수 $n$에 대하여, $n^2$ 이 3의 배수이면 $n$은 3의 배수임을 증명하라

pf)

자연수 $n$에 대해여서 3의 배수가 아닌 수를 나열하면 다음과 같다.
$3n+1$, $3n+2$

1)$3n+1$인 경우
$(3n+1)^2 = 9n^2 + 6n + 1$ 이므로 3의 배수가 아니다

2)$3n+2$인 경우
$(3n+2)^2 = 9n^2 + 12n + 1$ 이므로 3의 배수가 아니다

대우가 참이므로 원명제는 참이다.

## 집합과 조합론
